# Customer Churn Prediction (Machine Learning)

This project focuses on predicting customer churn using supervised machine learning techniques. The goal is to identify customers likely to leave a service based on behavioral and demographic features.

## Objectives / Methodology
- EDA
- Clean and preprocess
- Build model 
- Performance
- Results

## Dataset
The dataset contains customer information such as:
- Demographics
- Service usage
- Contract details
- Payment information
